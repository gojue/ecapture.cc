# 介绍

:::info eCapture旁观者无需CA证书，即可抓HTTPS/TLS明文数据 !

## eCapture旁观者是什么?
eBPF HOOK uprobe实现的各种用户态进程的数据捕获，无需改动原程序。

* SSL/HTTPS数据导出功能，针对HTTPS的数据包抓取，不需要导入CA证书。
* bash的命令捕获，HIDS的bash命令监控解决方案。
* mysql query等数据库的数据库审计解决方案。
